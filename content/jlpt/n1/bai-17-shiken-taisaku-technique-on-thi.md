**試験対策 (Shiken Taisaku)** N1 không chỉ về kiến thức mà về **strategic execution**. **Time management**, **question prioritization**, **stress control** và **mindset** - những yếu tố này quyết định **pass/fail** trong 180 phút áp lực cao.

## 1. Test Structure & Time Management

### 1.1 N1試験構成 (N1 Test Structure)

| Section | Time | Questions | Points | Strategy |
|---|---|---|---|---|
| **言語知識** | 110分 | | | |
| - 文字・語彙 | (~25分) | 33問 | 33点 | Quick execution |
| - 文法 | (~35分) | 32問 | 32点 | Pattern recognition |  
| - 読解 | (~50分) | 15問 | 35点 | Strategic reading |
| **聴解** | 70分 | 39問 | 60点 | Active listening |
| **Total** | 180分 | 119問 | 160点 | 100点以上 = Pass |

### 1.2 時間配分戦略 (Time Allocation Strategy)

**言語知識セクション (110分):**

| Part | Time | Strategy |
|---|---|---|
| **漢字読み** (1-6) | 5分 | Instant recognition or skip |
| **語彙選択** (7-21) | 15分 | Context clues + elimination |
| **語彙用法** (22-33) | 15分 | Collocation knowledge |
| **文法** (34-65) | 30分 | Pattern matching |
| **読解** (66-80) | 45分 | Strategic reading |

**Buffer Time:** 5分 for review/difficult questions

### 1.3 優先度マトリックス (Priority Matrix)

**High Priority (Must Get Right):**
- **Easy vocabulary** (7-15) → Quick points
- **Grammar patterns** you know → Confident answers
- **Short reading** passages → Time efficient

**Medium Priority:**
- **Challenging vocabulary** → Use elimination
- **Complex grammar** → Pattern recognition
- **Medium reading** → Strategic skimming

**Low Priority (Time Permitting):**
- **Very difficult kanji** → Guess intelligently
- **Unknown grammar** → Eliminate wrong answers
- **Long passages** → Focus on questions

## 2. Section-Specific Strategies

### 2.1 文字・語彙 Section Tactics

**漢字読み方 (1-6):**
- **Instant recognition** rule: 3秒以内
- **Unknown kanji:** Look for context clues
- **Multiple readings:** Choose most common
- **Don't overthink:** First instinct often correct

**語彙選択 (7-21):**
- **Read sentence completely** before options
- **Identify key context** words
- **Use elimination** method
- **Register matching:** formal sentence → formal vocabulary

**語彙用法 (22-33):**
- **Collocation knowledge** critical
- **Test each option** in context mentally
- **Common combinations** vs rare ones
- **Process of elimination**

### 2.2 文法 Section Mastery

**Pattern Recognition System:**

| Question Type | Recognition Cue | Strategy |
|---|---|---|
| **接続** (32-38) | Missing connector | Logic relationship |
| **語彙** (39-43) | Context/nuance | Meaning differentiation |
| **語順** (44-48) | Scrambled parts | Grammar rules |
| **文の組み立て** (49-54) | Sentence building | Systematic approach |
| **文章の文法** (55-65) | Passage grammar | Context reading |

**Elimination Techniques:**
1. **Obviously wrong** → eliminate immediately
2. **Grammar violations** → eliminate
3. **Meaning mismatch** → eliminate  
4. **Best fit** → select remaining

### 2.3 読解 Section Strategy

**Reading Approach by Type:**

**短文 (66-69):** 
- **Time limit:** 2分 per question
- **Strategy:** Read completely, then options
- **Focus:** Main message, not details

**中文 (70-74):**
- **Time limit:** 4分 per question  
- **Strategy:** Skim first, then targeted reading
- **Focus:** Paragraph main ideas

**長文 (75-80):**
- **Time limit:** 8分 per passage
- **Strategy:** Questions first, then strategic reading
- **Focus:** Overall structure and specific answers

**統合理解 (81-85):**
- **Time limit:** 10分 total
- **Strategy:** Compare sources systematically
- **Focus:** Common themes and differences

### 2.4 聴解 Section Excellence  

**Listening Strategy by Type:**

| Type | Duration | Strategy |
|---|---|---|
| **課題理解** | 1分30秒 | Focus on final instruction |
| **ポイント理解** | 1分30秒 | Listen for key message |
| **概要理解** | 3分 | Overall situation + attitude |
| **即時応答** | 10秒 | Quick reaction patterns |
| **統合理解** | 4分 | Multiple source integration |

## 3. Mental Strategies & Mindset

### 3.1 試験前マインドセット (Pre-Test Mindset)

**One Week Before:**
- **Review routine:** 基本パターンの確認
- **Confidence building:** 得意分野の復習  
- **Stress reduction:** 過度な新学習を避ける
- **Physical preparation:** 十分な睡眠

**Day of Test:**
- **Arrival:** 30分前に到着
- **Materials check:** 受験票・身分証明書・時計
- **Mental state:** リラックス but focused
- **Positive visualization:** 成功イメージ

### 3.2 試験中メンタル管理 (During-Test Mental Management)

**Pressure Management:**
- **Deep breathing:** 困った時は深呼吸
- **Time awareness:** 時計を定期的にチェック
- **Don't panic:** 分からない問題で止まらない
- **Move on strategy:** 30秒ルール

**Confidence Maintenance:**
- **Start with strengths:** 得意分野から
- **Build momentum:** 確実に取れる点を取る
- **Stay positive:** 一問のミスに引きずられない
- **Trust preparation:** 準備したことを信じる

### 3.3 各セクション間の切り替え (Section Transitions)

**Between Sections:**
- **Mental reset:** 前のセクションを忘れる
- **Strategy switch:** 次のセクション戦略に切り替え
- **Energy management:** 疲労を感じたら軽く伸び
- **Time check:** 残り時間の確認

## 4. Common Pitfalls & Avoidance

### 4.1 よくある失敗パターン (Common Failure Patterns)

**Time Management Failures:**
- **Perfectionism:** 一問に時間をかけすぎ
- **Panic mode:** 時間不足でパニック
- **Wrong priorities:** 難問に時間浪費
- **No time buffer:** 見直し時間なし

**Strategy Failures:**
- **No elimination:** 消去法を使わない
- **Context ignorance:** 文脈を読まない
- **Pattern blindness:** パターンに気づかない
- **Overconfidence:** 簡単と思って間違い

### 4.2 回避策 (Avoidance Strategies)

**Time Protection:**
- **Set timers:** セクション別タイマー
- **Progress checks:** 定期的進捗確認
- **Skip rule:** 30秒で分からなければスキップ
- **Return strategy:** 後で戻る問題にマーク

**Accuracy Improvement:**
- **Double-check:** 時間があるときは再確認
- **Context verification:** 文脈との整合性確認
- **Logic check:** 論理的に正しいか確認
- **Gut feeling:** 直感も大切にする

## 6. Score Calculation & Strategy

### 6.1 合格ライン (Passing Requirements)

**Requirements for N1 Pass:**
- **Overall score:** 100/160点以上 (62.5%)
- **Each section minimum:** 
  - 言語知識: 19/60点以上 (32%)
  - 聴解: 19/60点以上 (32%)

**Strategic Implication:**
- **Overall focus:** トータルで100点を目指す  
- **Section balance:** どちらも最低点をクリア
- **Compensation possible:** 得意分野で苦手分野をカバー

### 6.2 点数配分戦略 (Point Allocation Strategy)

**Conservative Strategy (安全重視):**
- **言語知識:** 55点目標 (90%)
- **聴解:** 45点目標 (75%)
- **Total:** 100点 → 合格

**Balanced Strategy (バランス型):**
- **言語知識:** 50点目標 (83%)  
- **聴解:** 50点目標 (83%)
- **Total:** 100点 → 合格

**Risk-Taking Strategy (チャレンジ型):**
- **Strong section:** 60点目標 (100%)
- **Weak section:** 40点目標 (67%)  
- **Total:** 100点 → 合格
